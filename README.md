# Project Calculator

A simple browser-based calculator built while learning JavaScript. It uses plain HTML, CSS, and JavaScript to handle basic arithmetic operations in a small, colorful interface.

## Live Demo

[View the calculator](https://shin890.github.io/Calculator/)

## Features

- Add, subtract, multiply, and divide numbers
- Calculate percentages
- Clear the display with one click
- Lightweight static project with no build tools or dependencies
- Beginner-friendly structure using separate HTML, CSS, and JavaScript files

## Preview

The calculator includes a display area, number buttons, operator buttons, a clear button, a percentage button, and a full-width equals button.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
Calculator/
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Start calculating.

No installation is required because this is a static web project.

## Note

This was one of my early JavaScript learning projects, so the goal was to understand the basics of DOM interaction and event handling.
